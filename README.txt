
# Casa Real

Um projeto web dedicado à apresentação e documentação da Casa Real.

## Descrição

Este repositório contém o código-fonte e documentação para o website da Casa Real, incluindo informações históricas, genealogia, e eventos.

## Características

- Apresentação institucional
- Informações históricas e genealógicas
- Galeria de eventos
- Design responsivo

## Requisitos

- Node.js (versão 14+)
- npm ou yarn
